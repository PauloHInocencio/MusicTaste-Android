# MusicTaste-Android

 MusicTast is an iTunes client App.

Functionalities
===========

- User can search for a music by name.
- User can discard (disliked) or save (like) the music information.
- User can see a list of the saved musicis. 


Knowledges applyeded
===========

* Networking Basics
* AsyncTask
* Fetching JSON
* Loopers
* Handlers
* HandlerThread
* SearchView
* Shared Preferences
